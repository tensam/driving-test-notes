# Opposite Direction Lane - Trailer Banned for Flow, Allowed for Works

## Question
Can a car pulling a trailer use a lane set up for travel in the opposite direction to usual?

## Options
- **A) Yes**
- **B) No, when it has been set up to improve traffic flow** ✅
- **C) Yes, except in the event of roadworks** ❌ (my answer)

## Correct Answer
B — No, when it has been set up to improve traffic flow.

## Why I Got It Wrong
选反了。教材 p.112 反方向车道规则：
- 疏导流量 → 仅限摩托车 + 不带拖车的小客车 → 拖车不能走
- 施工 → 所有车辆可通行 → 拖车能走

## Key Rule
| 反方向车道开通原因 | 谁能用 | 拖车 |
|----------------|--------|------|
| 疏导交通流量（flow/jam） | 摩托车 + 不带拖车的小客车 | ❌ |
| 施工（works） | 所有车辆 | ✅ |

记忆：疏导堵车→只许小车快速通过→拖车不行。施工→没别的路→全放行。

## Key Vocabulary
- opposite direction to usual = sentido contrario al habitual = 反方向车道
- traffic flow = flujo de tráfico = 交通流量
- roadworks = obras = 施工
