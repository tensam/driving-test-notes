# Cyclists Riding in Single File

## Question
Is it correct for this group of cyclists to ride on the right-hand side of the road, in single file?

## Options
- A) Yes, but they could take up the whole right-hand lane. ❌ (my answer)
- **B) Yes.** ✅
- C) No.

## Correct Answer
**B) Yes.**

## Explanation
Cyclists must ride on the right-hand side of the road, as close to the edge as possible, in single file. They are NOT allowed to take up the entire right-hand lane. The simple "Yes" is correct because the question only asks whether riding on the right in single file is correct - and it is.

## Key Rule
Cyclists ride **as far right as possible, single file**. They cannot occupy the full lane (option A adds an incorrect permission).
