# Overtaking in Curve - Allowed If Not Invading Opposite Lane

## Question
Can we overtake in a curve without visibility if the lanes are clearly marked and we do not invade the opposite lane?

## Options
- **A) Yes** ✅
- **B) No, since we do not have visibility** ❌ (my answer)
- C) Only if there are no vehicles coming from the opposite direction

## Correct Answer
A — Yes.

## Why I Got It Wrong
看到"弯道+无能见度"就条件反射选"不能超车"，忽略了题目给出的限定条件正好触发了例外条款。

## Key Rule
教材 p.135 超车禁止的**例外**：

> "Can be done, if the lanes are clearly delimited and I do it without invading the opposite direction, leaving the regulated lateral distance."

弯道禁止超车的本质原因 = 需要占用对向车道 + 看不见来车 = 危险。
如果满足：①车道分明 ②不侵入对向车道 ③保持横向距离 → 危险源不存在 → 可以超车。

## Key Vocabulary
- overtake = adelantar = 超车
- curve without visibility = curva sin visibilidad = 无能见度弯道
- clearly delimited lanes = carriles claramente delimitados = 明确分隔的车道
- lateral distance = distancia lateral = 横向距离
