# O2 Caravan ITV - Exempt First 6 Years ⚠猜对需复习

## Question
If you register an O2 category caravan today to be towed by a car, how often should you submit it to a periodic technical inspection for the first six years?

## Options
- **A) It is exempt during the first six years** ✅ (guessed)
- **B) Every three years**
- **C) Every two years**

## Correct Answer
A — Exempt during the first six years.

## Why I Need to Review
猜对但不确定。O2 = 750-3500kg 的拖车/房车，前6年免ITV年检。

## Key Rule
教材 p.39 ITV 年检周期：

| 车型 | 首次年检 | 之后 |
|------|--------|------|
| 小汽车/房车(自行) | 第4年 | 每2年→10年后每年 |
| 2轮小摩托 | 第3年 | 每2年 |
| 货车(≤3500kg) | 第2年 | 每2年→6年后每年→10年后每半年 |
| 轻型拖车 O1(≤750kg) | 不需要 | 不需要 |
| **重型拖车 O2(750-3500kg)** | **前6年免** | 按周期 |

## Key Vocabulary
- ITV = Inspección Técnica de Vehículos = 车辆技术检查（年检）
- O2 category = 750-3500kg 拖车类别
- caravan = 房车（拖车式）
